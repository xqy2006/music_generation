## 使用方法
#### 1. 登录/注册你的Github账户
#### 2. 点击Issues
![method](https://xqy2006.github.io/store/0.png)
#### 3. 点击New issue
#### 4. 点击Get started
#### 5. 在输入框内填写要谱曲的歌词，点击Submit new issue
#### 6. 此时页面会跳转至issue详情页，系统需要一些时间处理请求，等待3-5分钟后刷新该页面，即可看到结果
